# Task Manager com Kanban

Este projeto é um gerenciador de tarefas estilo Trello simplificado.

### O que você vai praticar
- Componentes Livewire com drag-and-drop (usando SortableJS)
- Relacionamentos Eloquent (`hasMany`, `belongsTo`)
- Policies e autorização por usuário
- Eventos e listeners no Livewire

### Diferencial para o Portfólio
Todo mundo faz to-do list, mas um Kanban com Livewire já mostra domínio real de reatividade.
